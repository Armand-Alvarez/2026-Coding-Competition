# 07 - Stop gninnipS My sdroW!

## Problem Statement


_Write a function that takes in a string of one or more words, and returns the same string, but with all words that have five or more letters reversed (just like the name of this kata). Strings passed in will consist of only letters and spaces. Spaces will be included only when more than one word is present._


## Input Format: 
- String

## Output Format:
- String
    
## Constraints:
- None

## Sample:

| Input | Output |
| ----- | ---------------------- |
| Stop Spinning My Words  | Stop gninnipS My sdroW |


### Explanation:

None
