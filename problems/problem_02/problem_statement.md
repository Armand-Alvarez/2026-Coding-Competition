# 02 - Remove First and Last Character

## Problem Statement


_Your goal is to write a function that removes the first and last characters of a string. You are given one parameter, the original string._

_Important: Your function should handle strings of any `length >= 2` characters. For strings with exactly `2` characters, return an empty string.


## Input Format: 
- String

## Output Format:
- String
    
## Constraints:
- Input string will always be at least 2 characters
- Function should handle strings containing letters, numbers, and special characters.

## Sample:

| Input      | Output   |
| ---------- | -------- |
| 'eloquent' | 'loquen' |
| 'ab'       | ''       |
| 'xyz'      | 'y'      |
